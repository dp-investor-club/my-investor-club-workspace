# My Investor Workspace

**이 레포 전체가 여러분 것입니다.** DP Investor Club 커리큘럼을 수행하는 여러분만의 투자 판단 작업실이고, 클럽이 만든 교재(`club/`)와 도구(스킬)가 이미 들어 있습니다.

## 시작하기

이 폴더에서 `claude`를 실행하고, 아래를 입력하세요:

```
/start
```

AI가 작업실 구조를 안내하고, 여러분의 CLAUDE.md 시드를 인터뷰로 채워줍니다.

## 구조

```
├── CLAUDE.md      # 나의 기준 — 매주 케이스 판단에서 한 줄씩 자랍니다
├── context.md     # 지금 진행 중인 케이스에 대한 나의 판단 상태
├── journey.md     # 클럽 여정 로그 — 세션 노트, 과제 상태, 커리어 질문
├── cases/         # 케이스 작업 (내 것)
├── sources/       # 소스 노트 (내 것)
├── outputs/       # 주차별 산출물 (내 것)
├── club/          # 클럽 교재 — 지식 카드, 케이스팩, 주차 자료 (읽기 전용)
└── .claude/skills # 도구 — /start /concepts /harness-check /submit ...
```

## 자주 쓰는 것

| 하고 싶은 것 | 입력 |
|---|---|
| 처음 세팅 | `/start` |
| 모르는 용어 | 그냥 물어보세요: "MCP가 뭐야?" (또는 `/concepts`) |
| 다음 수업 예습 | `/prep-week2` |
| 자가진단 | `/harness-check` |
| 과제 제출 | `/submit` |
| 교재 업데이트 | "club 교재 최신으로 받아줘" |

> 이 레포의 모든 기업 사례는 가상이며, 어떤 내용도 투자 조언이 아닙니다.
