# My Investor Club Workspace

**이 레포 전체가 여러분 것입니다.** DP Investor Club의 몇 달 — 강연 10회, 모의투자, 팀 프로젝트, 커리어세션, 멘토링, 캠프 — 을 AI와 함께 수행하는 여러분만의 조종석이고, 클럽이 만든 교재(`club/`)와 도구(스킬)가 이미 들어 있습니다.

## 시작하기

이 폴더에서 `claude`를 실행하고:

```
/start
```

## 구조 — 클럽의 레인이 곧 폴더입니다

```
├── CLAUDE.md       # 나의 기준 — 매주 판단에서 한 줄씩 자랍니다
├── context.md      # 나의 모의투자 판단 상태
├── journey.md      # 클럽 전체 지도 — 7개 레인, 일정, 세션 노트
├── lectures/       # 강연 사전 조사 + 강연 노트 (10회)
├── mock-invest/    # 모의투자 판단 기록 (매주)
├── team-project/   # 팀 프로젝트 산출물
├── career/         # 커리어세션·멘토링 질문과 기록
├── sources/        # 소스 노트
├── outputs/        # AI Intensive 주차별 산출물
├── club/           # 클럽 교재 — 지식 카드 등 (읽기 전용)
└── .claude/skills  # 도구
```

## 매주의 리듬

| 언제 | 무엇 | 입력 |
|---|---|---|
| 강연 전 | 연사 사전 조사 준비 | `/lecture-prep` |
| 모의투자 전후 | 판단 기록·업데이트 | `/mock-invest-log` |
| AI 수업 전 | 예습 랩 (30분) | `/prep-week2` (주차별) |
| AI 수업 후 | 과제 제출 | `/submit` |
| 아무 때나 | 모르는 용어 | 그냥 물어보세요 ("MCP가 뭐야?") |
| 주 1회 | 자가진단 | `/harness-check` |

교재 업데이트가 필요하면: "club 교재 최신으로 받아줘"

> 이 레포는 교육 훈련용입니다. 어떤 내용도 투자 조언이 아니며, 모의투자 기록은 실제 매매 판단이 아닙니다.
