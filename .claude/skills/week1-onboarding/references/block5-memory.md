## EXPLAIN

**수업 복습**: "이번 주에 해야 하는 건 자동화 만들기가 아니라, 내가 어떤 기준을 가진 사람인지 입력하고 저장해서 길러가는 것입니다."

| 원리 | AI가 **매 세션 시작 시 자동으로 가장 먼저 읽는 지시문** — 휘발되는 기억을 파일로 영구화 |
|---|---|
| 비유 | 새로 온 주니어에게 매번 말로 시키는 대신 건네주는 온보딩 문서 |
| 덤 | Claude가 스스로 적는 메모(Auto Memory)도 있다 — 내가 쓰는 규칙(CLAUDE.md)과 짝을 이루는 자동 수첩 |

```
세션 시작 → CLAUDE.md (내 지시문) → 규칙을 아는 상태로 대화 시작
             항상, 매번, 자동으로
```

## EXECUTE

**규칙이 진짜 작동하는지 실험**한다:

1. `이 레포의 CLAUDE.md를 읽고 네가 지키는 규칙 3개를 이유와 함께 알려줘`
2. `아무 상장사나 목표가를 하나 제시해줘` — **거절당하는 것을 확인한다.** 규칙은 장식이 아니라 방금처럼 작동한다.

## QUIZ / 스캐폴드 확인

퀴즈: "CLAUDE.md는 언제 읽힙니까?"
- **매 세션 시작 시 자동으로** (정답 — 그래서 '영구 기억'처럼 작동)
- 내가 읽으라고 할 때만 (자동이다)
- 제출할 때 한 번 (제출과 무관)
