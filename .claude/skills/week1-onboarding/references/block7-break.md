# Block 7: 쉬어가기 — 내 작업실에 문패 달기

> 퀴즈 없는 블록. Phase A만 진행하고, "완료"가 오면 Block 8로.

## EXPLAIN

여기까지 오느라 수고했다. 잠깐 숨 고르기 — 재미있는 것 하나.

Claude Code 화면 맨 아래 **상태 표시줄(status line)**을 내 마음대로 꾸밀 수 있다. 지금 쓰는 모델, 컨텍스트 사용량 같은 것을 실시간으로 보여주는 나만의 계기판이다. 한 줄이면 된다 — Claude가 알아서 스크립트를 만들고 설정까지 해준다.

## EXECUTE

Claude Code에 그대로 입력:

```
/statusline 모델 이름과 컨텍스트 사용률을 프로그레스 바로 보여줘
```

결과 예: `[Fable] ▓▓░░░░░░░░ 25%`

마음에 안 들면 다시 시키면 된다 — "이모지 넣어줘", "브랜치도 보여줘". **여러분 작업실의 첫 인테리어입니다.** 원하는 모양이 나오면 다음으로.
