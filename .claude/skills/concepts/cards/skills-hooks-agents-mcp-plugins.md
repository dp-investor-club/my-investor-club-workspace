# AI 도구 배치 — Skills · Hooks · Agents · MCP · Plugins

> 5종 컴포넌트는 각각 다른 역할을 합니다. 헷갈리면 "누가 무엇을 담당하는 팀인가"로 기억하세요.

## 역할 분담

| 컴포넌트 | 비유 | 역할 | 예 |
|---|---|---|---|
| **Skills** | 업무 매뉴얼 | 반복 작업의 절차를 레시피로 — `/이름`으로 실행 | 이 레포의 `/onboarding`, `/submit` |
| **Hooks** | 자동 안전장치 | 행동 전(차단)·후(검사)·종료(일지)에 자동 실행 | 커밋 전 크레덴셜 검사 |
| **Agents** | 전문가 팀원 파견 | 별도 AI에게 하위 작업을 맡기고 결과만 수신 | "이 공시 3건 각각 요약해와" |
| **MCP** | 외부 도구 조작 권한 | AI를 외부 시스템에 연결 — 읽고 쓰게 | 공시 DB, Slack, 뉴스 피드 연결 |
| **Plugins** | 확장팩 | 위 전부를 패키지로 묶어 배포·공유하는 단위 | **이 레포 자체** (`.claude-plugin/`) |

핵심 구분 한 줄: **Skill은 "어떻게 할지"(절차), MCP는 "무엇에 접근할지"(연결), Plugin은 "어떻게 나눠줄지"(포장)**입니다.
