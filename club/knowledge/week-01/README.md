# knowledge/ — 개념 카드

**카드 1장 = 개념 1개.** 참가자가 "MCP가 뭐야?", "스킬이 뭐야?" 같은 질문을 하면, AI는 이 폴더의 카드를 읽고 카드의 비유와 근거로 답합니다 (`/concepts` skill 참조).

## 카드 규칙

- 모든 카드는 **투자자/실무자 비유**를 먼저, 기술 정의를 나중에 둔다.
- 모든 카드는 **"직접 해보기"** 한 가지로 끝난다 — 개념은 눈에 보이는 실행에 붙어야 남는다.
- 카드는 주차별 폴더에 들어간다. 아직 안 열린 주차의 개념은 지도만 보여주고 상세는 그 주에 연다.
